Platform Settings

Manage global configurations, site status, and banners.

Maintenance Mode

When active, the site will be temporarily disabled for regular users. Only admins will be able to log in and bypass the maintenance screen.

Allow New Registrations

Toggle this off to temporarily prevent new users from signing up on the platform. Existing users can still log in.

Global Announcement Banner

Display a prominent banner across the top of the user dashboard to announce sales, new features, or important updates.
